# Transparent prints

- PETG transparent
  - Fill pattern: Aligned Rectilinear @ 100%
  - Layer height: 0,1 mm
  - Slightly hotter -> better
  - Slow print -> better  (e.g. 15mm/s) (very important)
  - Fill angle: 0
  - Horizontal shells:
    - Top: 0
    - Bottom: 0
  - Fan speed: 0

- White PP can be made translucent (not the same as transparent) with an automatic smoothing machine (Powerfuse S)
